### 1. Initiate a New Support Ticket
Navigate to the **Support Center** – **Support Ticket System** and select “**Open a New Ticket**” to begin the process.

![Screenshot](https://github.com/alexandrpaul/ost-tlcex/blob/751c6cab5b475d0a4e3ac280cb2ef0ac1c4c7f6f/Screenshots/img1.png)



### 2. Ticket Submission Overview
When a new ticket is opened, the interface displays **Contact Information**, **Help Topic**, and **Ticket Details**. End users provide the necessary information regarding their support request. In this scenario, the user **Lorin Gutierrez** requires a **password reset** due to a forgotten credential.

![Screenshot](https://github.com/alexandrpaul/ost-tlcex/blob/751c6cab5b475d0a4e3ac280cb2ef0ac1c4c7f6f/Screenshots/img2.png)



### 3. Password Reset Ticket Review
Upon submission of the password reset request, the Support Desk Technician is presented with the ticket details. The information submitted under the user account Lorin is displayed for verification and action.

![Screenshot](https://github.com/alexandrpaul/ost-tlcex/blob/751c6cab5b475d0a4e3ac280cb2ef0ac1c4c7f6f/Screenshots/img3.png)



### 4. Ticket Update Summary
The ticket was updated and assigned to Test, SLA adjusted, and priority level modified. Following these updates, a response was sent to the requester confirming that the password reset is scheduled.

![Screenshot](https://github.com/alexandrpaul/ost-tlcex/blob/751c6cab5b475d0a4e3ac280cb2ef0ac1c4c7f6f/Screenshots/img4.png)



### 5. Password Reset Completion
Following a successful password reset for Lorin, Test notifies the user and formally closes the support ticket.

![Screenshot](https://github.com/alexandrpaul/ost-tlcex/blob/751c6cab5b475d0a4e3ac280cb2ef0ac1c4c7f6f/Screenshots/img5.png)
